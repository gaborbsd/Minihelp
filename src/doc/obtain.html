Obtaining Minihelp

Minihelp is currently only distributed as source code. The project is hosted at GitHub and the repository can be located at https://github.com/gaborbsd/Minihelp. You can download the sources as a zip file from this URL, or you can clone the repository from command-line, using the following command:

git clone https://github.com/gaborbsd/Minihelp.git

If you are unfamiliar with the git version control system, you may want to take a look at the Getting Started section of the documentation.
